# 📄 PDF Reader & Analyzer RAG Agent

A beginner-friendly **Retrieval-Augmented Generation (RAG)** project built to explore the fundamentals of **Large Language Models (LLMs)**, vector embeddings, semantic search, and AI-powered document question answering.

The application allows users to upload a PDF, processes the document into searchable embeddings, retrieves the most relevant context, and generates accurate responses using an LLM.

---

## 🚀 Features

- 📄 Upload and analyze PDF documents
- ✂️ Automatic text extraction and chunking
- 🧠 Generate vector embeddings
- 🔍 Semantic search for relevant document context
- 🤖 LLM-powered question answering
- ⚡ Fast and context-aware responses

---

## 🛠️ Tech Stack

- Python
- LangChain
- OpenAI / Gemini
- FAISS / ChromaDB
- PyPDF
- Streamlit

---

## ⚙️ How It Works

1. Upload a PDF document.
2. Extract text from the document.
3. Split the text into manageable chunks.
4. Convert each chunk into vector embeddings.
5. Store the embeddings in a vector database.
6. Retrieve the most relevant chunks based on the user's query.
7. Generate a response using the retrieved context and an LLM.
